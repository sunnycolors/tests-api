# tests-api — Montée en compétence QA

Suivi de mon apprentissage des tests API, de Postman aux pipelines CI/CD, en passant par l'automatisation JavaScript.

---

## Objectif professionnel

Devenir **QA Engineer** maîtrisant les tests API de bout en bout — tests manuels, automatisation, intégration CI/CD — avec une stack JavaScript moderne utilisée en entreprise.
